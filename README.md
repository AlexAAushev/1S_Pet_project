# Конфигурация предприятия производителя мебели
На чистой конфигурации создано решения для ООО МебельДомТорг, торгующей мебелью. Добавлены все необходимые справочники, документы, регистры, отчеты и др.

## Установка

База данных для платформы 1С:Предприятие 8.3 в формате .dt
